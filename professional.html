People with no experience type by looking at their keyboard rather than using their muscle memory using a few random fingers. This process limits the typing potential of the individual and is extremely slow compared to touch typing. The main objective in this case is to build a foundational muscle memory which makes the fingers used to the location of each key on the keyboard.
